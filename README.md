## Requirements:

1. web server (apache)
2. php > 7.0
3. mysql (v8 recommended)


## How to start:

1. run ```php artisan migrate --seed```
2. run ```php artisan admin:make```
3. run ```php artisan key:generate```
4. run ```php artisan passport:install```
6. run `composer dump-autoload`
